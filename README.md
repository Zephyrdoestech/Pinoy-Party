# Pinoy Party 🎉

**Pinoy Party** is a multiplayer Godot 4 party game inspired by classic Filipino street games! Gather your friends, roll the dice on the virtual board, and compete in fast-paced minigames to see who will be crowned the ultimate champion.

## 🎮 Features

* **Multiplayer Board Game:** Up to 4 players can join the same lobby and take turns rolling the dice, moving across the board, and triggering events.
* **Classic Filipino Minigames:** 
  * **Langit Lupa:** Survive the rising flood by reaching the safe platforms in time! The last player standing (or the first to the top) wins.
  * **Luksong Baka:** Time your jumps perfectly to clear the sweeping hurdle. It gets faster every round!
  * **Sack Race:** Mash your keys to hop to the finish line before your friends.
  * **Sari-Sari Store Trivia:** Test your knowledge with Pinoy trivia questions.
* **Smart Local & LAN Play:** Play with friends over the same Wi-Fi using a simple 5-letter room code. It even supports local testing on a single PC automatically!

## 🚀 Getting Started

### Prerequisites
* **Godot Engine 4.2+** (if you want to run from source)

### Running the Game
1. Clone this repository to your local machine.
2. Open the Godot Engine and import the `pinoy-party/project.godot` file.
3. Press **F5** (or click the Play button) to run the game!

### How to Play Multiplayer
1. One player clicks **Host Game** on the main menu. They will be given a random 5-letter Room Code.
2. Other players click **Join Game** and enter that exact 5-letter Room Code (and their chosen username).
3. The host can start the game once everyone is in the lobby!

*Note: For the room code scanner to work, all players must be connected to the same Local Area Network (Wi-Fi). If you are testing two windows on the same PC, the code will still work seamlessly!*
